Design Manifesto

Best Practices

Whitespace & Layout

Use whitespace to...

  • Indicate a focus
  • Create a hierarchy
  • Cluster certain items together
  • Create "breathing room"
Color

Use...

  • 1-4 colors
  • Shades of the same colors
  • Colors to communicate messages
  • Colors with high contrast for readability
Fonts & Text

Use...

  • Minimal text
  • 1-2 fonts
  • Large fonts
  • Line-height that promotes readability
  • Font weights to show hierarchy
Images

Use...

  • High-quality images
  • Images with consistent lightness, saturation, contrast, & hue
  • Consistent sizes
  • Consistent styles
